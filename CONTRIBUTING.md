# Contributing

## Scope

This repository is intended for benchmark release, maintenance, and reproducibility work around Bloom-Eval.

## Expected Contributions

- evaluation scripts
- prompt exports
- metadata cleaning
- result format validation
- documentation fixes

## Rules

- Do not commit copyrighted PDFs unless redistribution rights are confirmed.
- Keep prompt files machine-readable in `prompts/`.
- Keep benchmark metadata deterministic and versioned.
- Update `docs/release_checklist.md` when a release blocker is resolved.
